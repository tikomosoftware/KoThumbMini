# KoThumb Mini - 画像一括リサイズ・変換ツール (Mini版)

KoThumb Miniは、ドラッグ＆ドロップだけで画像を素早くリサイズ・変換できる、究極にシンプルなデスクトップアプリです。
「設定を開く手間さえ省きたい」という時に最適です。

## ✨ 特徴

- **究極のシンプル操作**: ウィンドウに画像をドロップした瞬間に処理が始まります。
- **高速並列処理**: マルチコアCPUをフル活用し、大量の画像もあっという間に処理。
- **モダンデザイン**: グラスモーフィズムを採用した美しいUI。
- **自動保存**: 元のファイルの隣に `resized` フォルダを作成して自動保存。

## 📸 機能

- **リサイズ**: スライダーまたは数値入力で長辺のサイズを指定可能。
- **形式変換**: JPG, PNG, WebP への一括変換。
- **プリセット**: アイコン作成(16px〜)からフルHDまで、よく使うサイズをメニューから即座に選択可能。
- **多言語対応**: OSの言語設定に合わせて自動的に日本語/英語が切り替わります。

## 🚀 使い方

### 📦 インストール
本パッケージは **.NET同梱版** です。
.NETランタイムが実行ファイルに含まれているため、追加のインストールは不要です。

1. ZIPファイルを展開
2. `KoThumbMini.exe` を実行

### 基本操作
1. **サイズを決める**: スライダー、数値入力、またはプリセット（）からサイズを選択
2. **形式を選ぶ**: JPG / PNG / WebP を選択
3. **ドロップ！**: 画像をウィンドウにドラッグ＆ドロップすると処理が始まり、完了後にフォルダが自動的に開きます。
