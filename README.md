# KoThumb Mini

KoThumb Miniは、さらにシンプルに、さらに美しくなった画像一括リサイズ・変換ツールです。
フォルダを指定する手間さえ省き、「ドロップして終わり」の体験を提供します。

## ✨ 特徴

- **究極のシンプルさ**: ウィンドウに画像をドロップするだけで処理が始まります。
- **モダンデザイン**: ダークモードとグラスモーフィズムを採用した、デスクトップに馴染む美しいUI。
- **高速並列処理**: .NET 9.0とImageSharpをフル活用。
- **自動保存**: 元のファイルの隣に `resized` フォルダを作成して自動保存します。

## 🚀 使い方

1. `KoThumbMini.exe` を起動します。
2. リサイズしたい画像（またはフォルダ）をウィンドウにドラッグ＆ドロップします。
3. 処理が終わると、自動的に出力フォルダが開きます。

## 💻 開発情報

- **Language**: C# 13 / .NET 9.0
- **UI Framework**: WPF
- **Image Engine**: SixLabors.ImageSharp
